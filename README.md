# Three.js Portfolio

🚀 A personal developer portfolio built using **Three.js** and modern **WebGL** techniques.

This project showcases interactive 3D elements, animations, and frontend experimentation using JavaScript and Three.js.

---

## 🔗 Live Demo
👉 https://sarangk2024.github.io/threejs/

---

## 🛠️ Tech Stack
- Three.js
- JavaScript (ES6+)
- HTML5
- CSS3
- WebGL

---

## 📌 Features
- Interactive 3D scene
- Smooth animations and transitions
- Lightweight and fast loading
- Fully client-side (no backend)
- No advertisements
- No user authentication
- No data collection

---

## 🔒 Security & Privacy
- This site is **static** and hosted on **GitHub Pages**
- No cookies are used
- No user data is collected or stored
- No tracking scripts
- No third-party analytics
- All scripts are loaded securely over **HTTPS**

If your browser shows a security warning, it is a **false positive** caused by WebGL / Three.js behavior in some security scanners.

## 🚀 Deployment
This project is deployed using **GitHub Pages**.

To deploy:
1. Push code to `main` branch
2. Go to **Settings → Pages**
3. Select source branch
4. Save
